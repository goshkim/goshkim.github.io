JavaScript에는 7개의 기본 데이터 타입이 있으며, 이들은 크게 원시 타입(Primitive Type)과 객체 타입(Object Type)으로 나눌 수 있습니다.

### 원시 타입

1. **숫자 (Number)**: 정수와 부동 소수점을 모두 포함하는 숫자 데이터 타입입니다.

```javascript
let age = 25;
```

2. 문자열 (String): 텍스트 데이터를 저장할 수 있습니다.

```javascript
let name = 'Alice';
```

3. 불리언 (Boolean): 참(true) 또는 거짓(false) 값을 가집니다.

```javascript
let isAdmin = false;
```

4. null: 명시적으로 값이 없음을 나타냅니다.

```javascript
let selectedItem = null;
```

5. undefined: 값이 할당되지 않은 변수를 나타냅니다.

```javascript
let notAssigned;
```

6. 심볼 (Symbol): 고유하고 변경 불가능한 값을 나타냅니다.

```javascript
let id = Symbol('id');
```

7. 객체 타입: **객체 (Object)**는 여러 개의 값(프로퍼티)을 포함할 수 있는 복합 데이터 타입입니다.

```javascript
let person = { name: 'Alice', age: 25 };
```
