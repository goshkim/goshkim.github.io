**static** 폴더에 있는 파일은 `/` 경로로 서빙되기 때문에

아래와 같이 블로그에서 그대로 불러올 수 있습니다.

```json
{
	"name": "kim"
}
```

```javascript
const f = () => {
	console.log('Hello World');
};
```

```java
public class Hi {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```

```typescript
interface User {
	name: string;
	age: number;
}
```
